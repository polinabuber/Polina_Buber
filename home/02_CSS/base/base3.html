Блок навигации

  • Ширина 15em
  • Должен присутствовать цвет фона
  • Располагается слева
  • Блок навигации наполняется контентом достаточным чтобы по высоте он начинал захватывать вторую статью

Статья 1

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех сатьях сделайте отступ снизу от соседей равным 2em, при помощи свойства margin-bottom.

Статья 2

Размер блока:
Занимает всю доступную ширину
Должен присутствовать цвет фона
Во всех статьях сделайте отступ снизу от соседей равным 2em, при помощи свойства margin-bottom. Картинка вставлена при помощи тэга, плавание организовано по правому краю внутри данной статьи этой картинки.

Подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
